# Tamu Travel Website

This project is a three-page travel website called Tamu Travel. The purpose of the website is to introduce travel experiences in Kenya, including safari adventures, beach holidays, and group travel packages.

The website contains three main pages:
- Home page (index.html) – introduces Tamu Travel and provides an overview of available travel experiences
- Our Services page (services.html) – describes safari tours, beach holidays, and group travel packages
- Book Now page (contact.html) – allows users to submit booking requests through a contact form

The website was created using HTML and CSS with a safari-themed colour scheme, consistent layout design, and clear navigation between pages.

## How to open the website

Download or clone the project folder, then open the file **index.html** in any web browser to view the website locally. From the home page, users can navigate to the Our Services and Book Now pages using the navigation menu.
